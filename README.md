# AI Student Performance Predictor

A FastAPI + Machine Learning + Groq AI project.

## Features

- Predict student pass/fail
- AI-generated study advice using Groq Llama 3.3
- Frontend using HTML, CSS, JavaScript
- FastAPI backend
- Machine Learning model trained on student performance dataset

## Tech Stack

- FastAPI
- Python
- Scikit-learn
- Groq API
- HTML/CSS/JavaScript

## Run Project

```bash
pip install -r requirements.txt
python train.py
fastapi dev app.py
```

Open:

http://127.0.0.1:8000
